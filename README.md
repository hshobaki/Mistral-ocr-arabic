# ميسترال OCR برو | Mistral OCR Pro 🚀

تطبيق ويب متطور مدعوم بالذكاء الاصطناعي لاستخراج النصوص من الصور وملفات PDF، مع ميزات تحليل ذكية تشمل التلخيص والترجمة الاحترافية.

## ✨ المميزات الرئيسية

- **استخراج فائق الدقة (OCR):** استخدام نموذج `mistral-ocr-latest` لدعم مثالي للغة العربية والإنجليزية.
- **دعم ملفات متعددة:** معالجة الصور (JPG, PNG) وملفات PDF متعددة الصفحات.
- **تحليل الذكاء الاصطناعي:**
    - **التلخيص:** تحويل النصوص الطويلة إلى نقاط موجزة.
    - **الترجمة:** ترجمة احترافية فورية (عربي/إنجليزي).
- **واجهة مستخدم احترافية:**
    - دعم كامل للغتين (العربية والإنجليزية) مع تبديل الـ RTL/LTR.
    - وضع ليلي (Dark Mode) مريح للعين.
    - أدوات تحكم متقدمة (تكبير/تصغير المعاينة، وتكبير خط المحرر).
- **تصدير مرن:** تصدير النتائج إلى صيغ (PDF, Word, Text).
- **خصوصية وأمان:** تخزين مفتاح ال API محلياً في المتصفح.

## 🛠 التكنولوجيات المستخدمة

- **الذكاء الاصطناعي:** [Mistral AI API](https://mistral.ai/)
- **اللغات:** HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **المكتبات:**
    - `pdf.js` لمعاينة ملفات PDF.
    - `marked.js` لمعالجة الـ Markdown.
    - `Cropper.js` لتحرير الصور قبل المعالجة.
    - `Font Awesome` للأيقونات التفاعلية.

## 🚀 كيف تبدأ؟

1. قم بفتح ملف `index.html` في أي متصفح حديث.
2. انقر على أيقونة الإعدادات (الترس) في الأعلى.
3. أدخل مفتاح الـ API الخاص بك من Mistral AI.
4. ارفع ملفك وابدأ المعالجة!
